Binary Exponentiation Alg & Binary Multiplication Alg

Binary Exponentiation Alg 快速幂（二进制幂、平方幂）

  • 将幂分解:$x^a = x ^{(a)_{bin}} = x^{a1} * x^{a2} * x^{a3}$，让等价幂之间有倍数关系（因为是二进制），利用乘数的幂之间的关系，大幅简化运算（减少重复的计算）

  • 假设$x^n$ ，通常求幂方法是$x^n = x * x * x * … * x$ (n个x), 复杂度是O(n)。快速幂的做法是$x^n = x^n = x ^ {(n){bin}} = x^{(n1){bin}} * x^{(n2){bin}} * … * x^{(nn){bin}}$, 复杂度是O(logn)。

  • 带入例子，求x^15

    • ```java
      // $x^15 = x * x * … * x$ (15次)
      int pow(int x){
      for(int i = 0; i < 15; i++){
      x *= x;
      }
      return x;
      }

  • 和快速幂的思想差不多，$x * a = x * a_{bin} = x * a1_{bin} + x * a2_{bin} + x * a3_{bin}$
